了解var let const

1.作用域不同

var: function 语句

除了在函数中用var定义的变量不能在全局中调用，在其他区域用var声明的变量都可以在全局调用

let,const: block语句

let 与 const 声明后只能在该块级作用域下或更深的代码块里被调用

2.重复声明变量？

var: 容易覆盖定义过的变量的值，而let不允许在一个块级作用域下重复定义一个相同的变量，例：

const：不允许重新赋值，但是不意味着不能改变它(对象时)的值，例：
